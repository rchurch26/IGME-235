Friendly Fire

Friendly Fire is a puzzle, action game where you are a knight that tried to fight someone stronger than you and you lost, losing your sword in the process. Now you must escape a dungeon without attacking but by making your enemies attack each other. This game was created in C# using the monogame engine. For this project, I worked on enemy implementation and health increase after every level. Enemy implementation had many problems with loading them from the level file and putting them in the right place. For this issue, I worked with the groupmate that implemented the level file reader and we tackled enemy loading and implementation together. There are many levels in this game with many more features to come. I hope you enjoy.

Asteroids

Asteroids is a remake of the classic game of the same name. You control a ship in an asteroid field and you try to survive for as long as possible and try to get a high score. This game was made in JavaScript using the Pixi.js engine. The game doesn't have a bunch of irregular shapes for asteroids and a trianle for the ship but actual sprites instead. In the future, I'm thinking of adding different backgrounds instead of just the standard black and white background. A challenge I tackled was the movement of the ship since it is difficult to have a ship move based on where it is looking. With the help of a friend, I was able to figure out what I needed to add to the creation of the ship and move the ship correctly. I hope you enjoy.

Trivia

Trivia is a basic game of trivia that offers many different genres, difficulties, and up to fifty questions. A user can choose these different settings and challenge their knowledge in all areas or certain areas choosing between the difficulties of easy, medium, and hard. Users can also answer questions at any difficulty. This project was made in JavaScript and the toughest part was getting the data to load correctly. Carefully reading the API I used helped fix this issue and I was able to display questions and answers correctly. I plan on adding more CSS styles to the site so it can be much more presentable and fun to try out. I hope you enjoy.
